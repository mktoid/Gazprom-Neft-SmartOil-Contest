### Задача №1
Предсказать стартовый (в течение первых 6 месяцев работы скважины) дебит нефти. 

### Задача №2
Предсказать дебит нефти на добывающих скважинах в течение 6 месяцев после проведения ГРП (Гидравлический разрыв пласта).
